How It Works!

  1. 1. Copy the URL of the website you want to open.
  2. 2. Paste the copied URL into the input field.
  3. 3. Click "Add More" to include additional website URLs.
  4. 4. Repeat the above steps until you've added all the websites you want to open.
  5. 5. Click "Download File" to obtain your customized batch file.
  6. 6. Now, double-click the downloaded file to automatically open all the websites in your default web browser.
  7. 7. Once file is downloaded give permission to browser if it asks for any and allow the browser to keep the file.
  8. 8. Now just double click on the file you get and all your specified webpages will be opened automatically.
